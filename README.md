# Inbound

**Stack**: React 19 + TypeScript 5.9 + Vite 7.2
**Linter Included**: Yes
**TestEnv**: Vitest

## Requirements

- NodeJS v24

## Quick Start

1. Open project folder from any terminal
2. In project root, install NPM modules (run command): `npm i`
3. Start application: `npm start` (localhost:4200)
4. To run unit tests (vitest), please run `npm test`
